Latitude vs Temperature

The purpose of this project was to analyze how the weather changes as we get closer to the equator. Four elements were observed: Temperature, Wind Speed, Humidity and Cloudiness. Open Weather API was used along with python and JSON traversals to visualize weather of 573 cities around the world, then split into the northwern and southern hemispheres for a closer look. Here we will focus on the northern hemisphere. You can find the source data as well as the visualizations created during the analysis along with some summarized observations of the trends.
