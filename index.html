How's the Weather?

The purpose of this project was to analyze how the weather changes as we get closer to the equator. Four elements were observed: Temperature, Wind Speed, Humidity and Cloudiness.
Open Weather API was used along with python and JSON traversals to visualize weather of 573 cities around the world. You can find the source data as well as the visualizations created during the analysis along with some summarized observations of the trends.
